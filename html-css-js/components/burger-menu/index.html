Burger menu

CSS code

                .burger-menu {
                    background-color: black;
                    width: 40px;
                    height: 40px;
                    border: none;
                }
                .burger-menu::before {
                    content: "";
                    display: block;
                    border-top: solid 3px white;
                    border-bottom: solid 3px white;
                    padding: 2px 0;
                    transition: all .25s;
                }
                .burger-menu::after {
                    content: "";
                    display: block;
                    border-bottom: solid 3px white;
                    padding: 2px 0;
                    transition: all .25s;
                }
                .burger-menu:hover {
                    cursor: pointer;
                }
                .burger-menu.open::before {
                    border-bottom: 0;
                    transform: translateY(5px) rotate(45deg);
                }
                .burger-menu.open::after {
                    transform: translateY(-5px) rotate(-45deg);
                }

JavaScript code (with jQuery)

                const burger_menu = $('.burger-menu');
                burger_menu.click( function() {
                    $(this).toggleClass('open');
                });
